# 复杂画面 JSON 提示词生成器

## 📝 工具简介 (What is this tool?)
这是一个专为游戏开发者、美术设计和 Prompt 工程师打造的**复杂游戏画面 JSON 提示词（Prompt）可视化生成工具**。  
工具不仅提供了一个功能详尽的表单界面（涵盖从镜头、环境、光照到角色、战斗效果等数十个细分维度），还内置了 **AI 智能解析功能**（基于 Google Gemini AI）。用户可以一键上传优质参考截图，利用 AI 自动逆向解析画面内包含的各种游戏元素，并填充至结构化的字段中，**极大缩短编写复杂大场景提示词的时间**。

## ⚙️ 为什么使用 JSON 格式？ (Why JSON?)
在生成复杂的 AI 图像（特别是游戏大场景、多角色交互或特定的俯视角 SLG 时），传统的“写作文式”长文本提示词通常面临很大挑战：
1. **要素丢失与注意力稀释**：文本越长，AI 模型越容易“遗忘”后面的描述（比如限定好的护甲类型、特定的渲染光照等）。
2. **结构化控制**：JSON 强制规范了提示词的分类与层级关系（例如物理隔离 `Camera`（镜头透视） 和 `CombatEffects`（战斗特效））。这使得下游图像生成工作流（比如结合 ComfyUI、特定生图 API 节点解析）能够精准抽取局部特征并施加权重。
3. **沉淀与可复用化**：JSON 格式天生利于程序读取、修改和存储。团队可以建立标准化的“游戏资产提示词库”，遇到相似需求时，直接提取历史 JSON 数据复用或微调即可。

## 🎯 适合什么场景？ (Use Case Scenarios)
- **游戏概念研发（Demo 阶段）**：快速设定“暗黑风格ARPG/俯视角SLG”等细分品类的场景视觉标准与宣发视觉图。
- **竞品画面分析与逆向复刻**：上传优秀的同行游戏截图，通过“AI智能解析”一键拆解其美术构图、光影配置和环境材质，反向提炼出属于自己的结构化生图参数。
- **程序化批量资产生成**：配合外部自动化流水线或脚本引擎，可将生成的标准 JSON 数据丢入云端渲染程序，批量生成统一透视规格、统一打光的游戏宣传图或加载过场插画。

## 🚀 部署与运行方式 (Deployment)
为了团队内多人协同使用且**绝对保护 API Key 不被泄露**，本项目已配置为支持 Vercel Serverless 代理部署：

1. **Fork/Clone 本项目**：将代码仓库推送到你的个人 GitHub。
2. **导入 Vercel**：在 [Vercel](https://vercel.com/) 工作台新建项目，选择该 GitHub 仓库进行导入。
3. **配置环境变量**：在 Vercel 项目的 `Settings -> Environment Variables` 中添加密钥：
   - Key: `GEMINI_API_KEY`
   - Value: 你的 Google Gemini API Key
4. **一键分发**：点击 Deploy 部署完成后，Vercel 会生成一个安全的 `.vercel.app` 链接。将该链接发给团队成员，即可**免配置密钥直接使用**（图片解析请求会由后端安全转发，前端抓包无法看到真实 Key）。

> **💡 Note：** 
> 免费版 Vercel 对请求负载有限制，但本项目已内置**前端 Canvas 自动缩放压缩脚本**。用户上传的高清或超大游戏截图，在传输前会自动在浏览器端无损压缩，完美绕过 4.5MB 上传限制，稳定极速出词。
